# ZK for Artists Slides

Markdown slides introducing artists to Zero Knowledge cryptography and Trusted Execution Environments (TEEs).

Presented at the Glitch Artist Residency in November 2025.

## Contents

These slides cover:
- How ZK enables artists to prove human creative input vs AI generation
- Verifiable computation without revealing underlying data
- Leading zkVMs (SP1, RISC Zero, Jolt) with code examples
- TEEs vs zkVMs trade-offs
- The path forward for verifiable AI and creative processes

## Viewing as Slides

These markdown files are designed to be converted into slides using [HackMD](https://hackmd.io/).

To view as slides:
1. Copy the content from `zk-overview-slides.md`
2. Paste into HackMD
3. Use the slide mode (slides are separated by `---`)

## Format

Each slide is kept intentionally brief (1-2 sentences) for maximum impact during presentation.
